class Metric.evaluation_metrics(labels, input, parameters)[source]¶

Bases: object

“This class contains functions that can calculate
the evaluation metrices using the labels and the predictions of the model”
Accuracy(data_size)[source]¶

“calculate accuracy of training model” :param data_size: datasize of dataset(mnist, cifar-10) :return: accuracy of training model -> summation of True Positive / data_size

F1_score()[source]¶

“calculate F1_score of training model” :return: if(True Positive > 0) {2* (Precision * Recall / Precision + Recall)} else {0}

FN()[source]¶

“False Negative - Number of times the model predicts negative and the actual label is positive”

Returns:False Negative -> (summation of column of confusionMatrix) - list of True Positive
FP()[source]¶

“False Positive : Number of times the model predicts positive and the actual label is negative”

Returns:False Positive -> (summation of row of confusionMatrix) - list of True Positive
Precision()[source]¶

“calculate Precision of training model : What proportion of positive identifications was actually correct?”

:return:mean of (True Positive / True Positive + False Positive )

Recall()[source]¶

“calculate Recall of training model : What proportion of actual positives was identified correctly?” :return: mean(True Positive / True Positive + False Negative)

TP()[source]¶

“True Positive : Number of times the model predicts positive and the actual label is positive”

:return:True Positive -> list of diagonal of confusionMatrix

confusionMatrix()[source]¶

“function returns the confusion matrix that is used to calculate FP , TP and FN.”

Returns:confusion matrix to calculate accuracy, Precision, Recall, F1_score
